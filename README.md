# 📊 Past Paper Analyzer

## 🚀 Overview

Past Paper Analyzer is an AI-powered tool that helps students prepare smarter by analyzing previous years’ question papers. It identifies high-frequency topics and generates a prioritized study plan, allowing students to focus on the most important areas.

---

## ✨ Features

* 📂 Upload multiple past papers (PDF/TXT)
* 🔍 Automatic topic frequency analysis
* 📊 Identification of high-yield topics
* 📅 Smart study plan generation
* ⚡ Simple and clean user interface

---

## 🛠 Tech Stack

* **Frontend:** React.js
* **Backend:** Flask (Python)
* **Libraries:** pdfplumber (optional), Flask-CORS

---

## 🧠 How It Works

1. User uploads past question papers
2. System extracts text and scans for keywords
3. Topics are mapped and frequency is calculated
4. High-priority topics are identified
5. A smart day-wise study plan is generated

## ⚙️ How to Run Locally

### Backend

```bash
cd backend
pip install flask pdfplumber flask-cors
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm start
```
⚠️ Note

If the frontend folder preview is not visible on GitHub, please clone the repository to access the complete source code.
